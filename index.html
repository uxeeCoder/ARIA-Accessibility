ARIA-Accessibility

ARIA Functionality & Purpose

ARIA stands for Accessible Rich Internet Applications. It is a set of attributes that define ways to make web content and web applications more accessible to both machines and people with disabilities. It supplements HTML so that interactions and widgets commonly used in applications can be passed to assistive technologies.

ARIA-label functionality

There can be instances when the default accessible name of an element is not appropriate, use the aria-label attribute to define the accessible name of an element. For instance, request user feedback in an input field.

Use case of ARIA-hidden

ARIA hidden is used to hide the content from the screen readers. It can also be used to give access to hidden content only when a certain condition is met.

Section with Navigation role attribute

Navigation role is used to provide navigation to the corresponding content when a link to that content is clicked on. It can be helpful in cases where only keyboard navigation is possible.
